# 🤖 OpenAI Chatbot

A context-aware chatbot built with React and OpenAI's GPT API, integrated with Firebase backend for storing conversation history.

## 🔧 Tech Stack
- React.js
- OpenAI API
- Firebase (Firestore + Auth)
- Tailwind CSS

## 🚀 Features
- Real-time chat interface
- Conversation history saved in Firestore
- Context-aware replies
- Simple and clean UI
